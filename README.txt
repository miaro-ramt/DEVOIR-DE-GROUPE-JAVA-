# Exercices Java - Incrémentation et Tableaux

## Description
Ce projet contient des exercice en Java pour  l'apprentissage du langage.

Le programme permet de travailler sur :
- les incrémentation (pré-incrémentation et post-incrémentation)
- la manipulation de tableaux
- les tri et les recherche 

## Fichier principal
- ExercicesJava.java : contient le code principal de l'exercice

## Objectif
 Mieux comprendre le language .
